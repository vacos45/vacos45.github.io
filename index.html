Installation & Updates

While Kali Linux comes with Metasploit pre-installed, advanced users may wish to update or customize their installation.

  1. Update Repositories: 
    sudo apt update && sudo apt upgrade
  2. Install/Update Metasploit: 
    sudo apt install metasploit-framework
  3. Verify Version: 
    msfconsole --version

Basic Commands & Navigation

Launch Metasploit with msfconsole and try out these essential commands:

  • help – Displays all available commands.
  • search – Finds exploits and modules.
  • use – Selects a module (e.g., use exploit/windows/smb/ms17_010_eternalblue).
  • show options – Lists required parameters for the selected module.
  • set – Configures module options (e.g., set RHOSTS 192.168.1.100).
  • exploit – Executes the attack.

Advanced Techniques

Explore deeper functionalities of Metasploit with advanced modules and scripting.

Leverage auxiliary modules for comprehensive network scanning, vulnerability detection, and more. 

use auxiliary/scanner/portscan/tcp
set RHOSTS 192.168.1.0/24
run

After gaining access, post-exploitation modules help in maintaining persistence, gathering data, and escalating privileges. 

use post/windows/manage/enable_rdp
set SESSION 1
run
