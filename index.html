Installation & Updates

Although Kali Linux includes Metasploit, advanced users often need custom configurations and updates.

  1. Update Repositories: 
    sudo apt update && sudo apt upgrade
  2. Install/Update Metasploit: 
    sudo apt install metasploit-framework
  3. Verify Version: 
    msfconsole --version

Basic Commands & Navigation

Launch Metasploit with msfconsole and experiment with these commands:

  • help – Displays available commands.
  • search – Finds exploits and modules.
  • use – Selects a module (e.g., use exploit/windows/smb/ms17_010_eternalblue).
  • show options – Lists module parameters.
  • set – Configures module options (e.g., set RHOSTS 192.168.1.100).
  • exploit – Executes the attack.

Advanced Techniques

Explore deeper functionalities with advanced modules and custom scripting.

Use auxiliary modules for comprehensive network scanning and vulnerability detection. 

use auxiliary/scanner/portscan/tcp
set RHOSTS 192.168.1.0/24
run

Use post-exploitation modules to maintain access and escalate privileges. 

use post/windows/manage/enable_rdp
set SESSION 1
run
